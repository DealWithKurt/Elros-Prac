О сайте

Система учёта студентов

Этот сайт предназначен для управления информацией о студентах. Здесь вы можете просматривать, добавлять и отслеживать данные студентов.

Основные возможности:

  • Просмотр студентов - на главной странице в виде карточек
  • Табличное представление - удобная таблица со всеми данными
  • Добавление новых студентов - простая форма для внесения данных
  • Автоматическое сохранение - все данные сохраняются в файл
  • Уникальные идентификаторы - каждый студент получает свой ID

Как работать с сайтом:

  1. На главной странице вы видите список студентов в карточках. Каждая карточка содержит основную информацию: имя, возраст, ID и дату добавления.
  2. В таблице студентов представлены все данные в структурированном виде. Это удобно для быстрого просмотра и поиска информации.
  3. Для добавления нового студента перейдите в раздел "Добавить студента", заполните форму и нажмите кнопку "Добавить".
  4. Данные автоматически сохраняются в файл, поэтому при перезапуске приложения вся информация останется на месте.

Технические детали:

  • Backend: Python 3 с использованием фреймворка Flask
  • Frontend: HTML, CSS, Jinja2 шаблоны
  • Хранение данных: JSON файлы для простоты и наглядности
  • Валидация: Проверка корректности вводимых данных
  • Дизайн: Адаптивная фиолетовая тема

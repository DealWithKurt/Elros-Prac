# Система учета студентов

Веб-приложение для управления информацией о студентах на Flask.

## Функциональность

- Просмотр списка студентов (карточки и таблица)
- Добавление новых студентов с валидацией
- Редактирование данных студентов
- Удаление студентов с подтверждением
- Поиск студентов по имени
- Фильтрация по учебной группе
- Экспорт данных в CSV файл
- Сохранение данных в JSON файл

## Установка и запуск

### 1. Установите зависимости:

`pip install Flask`

2. Запустите приложение:

`python app.py`

3. Откройте в браузере:

`http://localhost:8585`


**Маршруты приложения**

    /	        GET	    Главная страница со списком студентов в виде карточек
    /students	GET	    Табличное представление студентов
    /add	        GET, POST   Форма добавления нового студента
    /edit/<id>	GET, POST   Форма редактирования студента
    /delete/<id>	GET	    Удаление студента с подтверждением
    /export/csv	GET	    Экспорт данных в CSV файл
    /about	        GET	    Страница "О сайте"

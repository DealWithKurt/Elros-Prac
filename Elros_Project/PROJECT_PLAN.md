# План проекта: "Учёт студентов"

## Цель проекта
Создать веб-приложение для учета студентов с возможностью:
- Добавления, просмотра, редактирования и удаления студентов
- Вычисления среднего балла
- Экспорта данных в CSV
- Фильтрации и поиска

## Сценарии работы пользователя

### Сценарий 1: Просмотр списка студентов
1. Пользователь заходит на главную страницу
2. Видит таблицу со всеми студентами
3. Может отсортировать по имени, возрасту или среднему баллу
4. Может найти студента по имени

### Сценарий 2: Добавление нового студента
1. Пользователь нажимает "Добавить студента"
2. Заполняет форму: имя, возраст, группа
3. Добавляет оценки по предметам
4. Сохраняет - студент появляется в списке

### Сценарий 3: Редактирование студента
1. В списке нажимает "Редактировать" у студента
2. Изменяет данные (имя, возраст, оценки)
3. Сохраняет изменения

### Сценарий 4: Удаление студента
1. В списке нажимает "Удалить" у студента
2. Подтверждает удаление
3. Студент исчезает из списка

### Сценарий 5: Экспорт данных
1. Пользователь нажимает "Экспорт в CSV"
2. Скачивает файл со всеми данными студентов

## Структура данных (JSON схема)

### Файл: `data/students.json`

    [
      {
        "id": "уникальный_идентификатор",
        "name": "Имя Фамилия",
        "age": 20,
        "group": "ИС-21",
        "grades": {
          "Математика": 4,
          "Физика": 5,
          "Информатика": 4
        },
        "created_at": "2026-02-04T10:30:00",
        "updated_at": "2026-02-04T10:30:00"
      }
    ]

### Поля:
1. id (string): уникальный ID студента
2. name (string): имя и фамилия
3. age (integer): возраст (16-35)
4. group (string): учебная группа
5. grades (object): словарь предмет→оценка (2-5)
6. created_at (string): дата создания
7. updated_at (string): дата последнего обновления

### Маршруты (Routes)
**GET маршруты:**
- / - главная страница, список студентов
- /student/<id> - страница студента (детальная информация)
- /add - форма добавления студента
- /edit/<id> - форма редактирования студента
- /about - информация о проекте
- /export - экспорт данных в CSV

**POST маршруты:**
- /add - добавление нового студента
- /edit/<id> - обновление данных студента
- /delete/<id> - удаление студента

**API маршруты (опционально):**
- /api/students - список всех студентов (JSON)
- /api/student/<id> - данные одного студента (JSON)
- /api/stats - статистика (JSON)

### Страницы (Templates)
1. base.html

Базовый шаблон с:
- Навигационным меню
- Заголовком сайта
- Областью для контента
- Футером
- Стилями (фиолетовая тема)

2. index.html

Главная страница с:
- Таблицей студентов
- Кнопками действий (добавить, экспорт)
- Поиском по имени
- Сортировкой
- Статистикой (общее количество, средний возраст)

3. student.html

Страница студента с:
- Полной информацией о студенте
- Списком оценок
- Средним баллом
- Кнопками редактировать/удалить

4. form.html

Общая форма для:
- Добавления нового студента
- Редактирования существующего
- Валидации данных

5. about.html

Страница "О проекте":
- Описание функционала
- Инструкция по использованию
- Информация о команде

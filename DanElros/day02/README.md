# День 2. Условия, циклы, функции, обработка ошибок.

## Структура проекта
- `calc.py` - модуль с базовыми арифметическими функциями
- `cli_calc.py` - интерактивный CLI-калькулятор
- `primes.py` - проверка простых чисел

## Запуск

### Калькулятор:

1. Запустите нужный скрипт:

# День 14. Введение во Flask: маршрут, шаблон, форма.

Простой веб-сайт на Flask для управления студентами.

## Описание

Веб-приложение с тремя страницами:
1. **Главная** - список студентов
2. **О сайте** - информация о проекте
3. **Добавить студента** - форма для добавления новых студентов

## Установка и запуск

### 1. Установите зависимости:

`pip install Flask`

# 2. Запустите приложение

`python app.py`

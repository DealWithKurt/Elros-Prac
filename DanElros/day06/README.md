# День 6. Исключения углублённо, логирование.

## Описание
Программа для безопасной обработки файлов с числами. Читает числа из файла, рассчитывает статистику и логирует все операции.

## Структура
- `safe_io.py` - модуль с функциями обработки файлов
- `cli_safe_io.py` - CLI интерфейс для запуска
- `logs/app.log` - файл логов (создается автоматически)

## Использование

### 1. Создать тестовый файл:

 - `python cli_safe_io.py --create-test numbers.txt`

### 2. Обработать файл:
 - `python cli_safe_io.py numbers.txt`

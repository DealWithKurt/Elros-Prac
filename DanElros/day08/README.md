# День 8. CSV — чтение, простая аналитика.

## Описание
Программа анализирует CSV файл с оценками студентов, рассчитывает статистику и сохраняет отчет.

## Структура
- `generate_test_data.py` - генерация тестового CSV файла
- `csv_stats.py` - анализ данных и генерация отчета
- `data/students_scores.csv` - исходные данные
- `reports/csv_stats.txt` - отчет с результатами

## Использование

### 1. Сгенерировать тестовые данные:

`python generate_test_data.py`

### 2. Проанализировать данные:

`python csv_stats.py`

# День 10. Мини-проект: «Статистика студентов».

## Описание
Проект для анализа оценок студентов из CSV файлов. Включает загрузку данных, анализ статистики, визуализацию и генерацию отчетов.
 
## Установка

# 1. Установите зависимости
`pip install matplotlib`

# 2. Убедитесь что находитесь в папке day10

`cd day10`

# 3. Проект готов к использованию

1. Создать тестовые данные:

`python main.py --create-test test.csv`

2. Проанализировать данные:

`python main.py data/test.csv`

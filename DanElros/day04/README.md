# День 4. Файлы, JSON, безопасная запись.

## Структура проекта
- `create_test_files.py` - скрипт для генерации тестовых файлов
- `read_numbers.py` - скрипт для чтения чисел и расчета статистики
- `json_roundtrip.py` - скрипт для работы с JSON данными
- `io_tasks/` - папка с входными и выходными файлами

## Подготовка к работе

### 1. Создание тестовых файлов:

   - `python create_test_files.py`

Создаст:
- `io_tasks/numbers.txt` - файл с 20-40 числами и тестовыми строками
- `io_tasks/students.json` - JSON файл с данными 10 студентов

### 2. Основные скрипты:

#### Чтение чисел и генерация отчета:

   - `python read_numbers.py

Что делает скрипт:
- Читает числа из `io_tasks/numbers.txt`
- Игнорирует пустые и некорректные строки
- Вычисляет минимум, максимум, среднее и количество чисел
- Сохраняет отчет в `io_tasks/result.txt` с атомарной записью

#### Работа с JSON:

   - `python json_roundtrip.py`

Что делает скрипт:
- Загружает данные из `io_tasks/students.json`
- Добавляет нового студента
- Сохраняет обновленные данные в `io_tasks/students_updated.json`
- Гарантирует, что исходный файл не изменен

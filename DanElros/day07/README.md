# День 7. HTTP-запросы и работа с простым внешним API.

## Описание
Программа генерирует задачи с русскими названиями, сохраняет в файл и анализирует статистику.

## Структура
- `fetch_todos.py` - генерация данных
- `todos_report.py` - анализ данных и отчет
- `data/todos.json` - сохраненные данные

## Использование

### Установите зависимости

`pip install requests`

### 1. Сгенерировать задачи:

`python fetch_todos.py`

### 2. Сгенерировать отчет:

`python todos_report.py`

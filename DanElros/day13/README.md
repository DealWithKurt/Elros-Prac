# День 13. Простейший GUI на Tkinter.

Простое приложение "Блокнот" с графическим интерфейсом.

## Описание

Простой текстовый редактор с возможностью:
- Создания новых текстовых файлов
- Открытия существующих файлов
- Сохранения текста в файлы
- Очистки текстового поля

## Установка и запуск

Tkinter обычно идет в комплекте с Python. Для запуска:

`python notepad.py`

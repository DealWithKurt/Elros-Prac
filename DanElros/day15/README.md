# День 15. Flask с сохранением в JSON.

Веб-приложение для управления студентами с сохранением данных в JSON файл.

## Описание

- Просматривать список студентов в таблице
- Добавлять новых студентов через форму
- Сохранять данные в JSON файл
- Данные сохраняются между перезапусками приложения

## Установка и запуск

### 1. Установите зависимости:

`pip install Flask`

# 2. Запустите приложение

`python app.py`

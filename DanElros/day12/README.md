# День 12. ООП — наследование и переопределение.

Демонстрация наследования, полиморфизма и переопределения методов.

## Классы

### Person (базовый класс)
- `__init__(name, age)` - конструктор
- `info()` - возвращает информацию о человеке

### Student (наследник Person)
- Добавляет поле `grades`
- `add_grade(grade)` - добавляет оценку
- `avg_grade()` - возвращает средний балл
- Переопределяет `info()` - выводит информацию о студенте с оценками
- Реализует методы сравнения (`__lt__`, `__eq__` и др.)

### Teacher (наследник Person)
- Добавляет поле `subject`
- Переопределяет `info()` - выводит информацию об учителе с предметом

## Использование

`python demo.py`

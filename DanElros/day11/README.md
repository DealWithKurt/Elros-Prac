# День 11. ООП — классы и методы.

Простая реализация классов Student и Group.

## Классы

### Student
- `__init__(name)` - создает студента
- `add_grade(grade)` - добавляет оценку (2-5)
- `avg()` - возвращает средний балл
- `as_dict()` - возвращает данные в виде словаря

### Group  
- `__init__(name)` - создает группу
- `add_student(student)` - добавляет студента
- `best(n=3)` - возвращает n лучших студентов
- `avg_all()` - возвращает средний балл по группе

## Использование

`python demo.py`

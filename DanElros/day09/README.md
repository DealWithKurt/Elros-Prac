# День 9. Первая визуализация.

## Описание
Программа строит графики по данным оценок студентов из CSV файла.

## Структура
- `plot_grades.py` - основной скрипт для визуализации
- `data/students_scores.csv` - данные об оценках
- `figures/` - папка с сохраненными графиками

## Использование

### Обязательно установите зависимости

`pip install matplotlib`

### 1. Построить графики:

`python plot_grades.py`
